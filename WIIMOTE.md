# Setting up the Wii Remote support

- Install XWiimote:
$ sudo apt-get xwiimote

- Build + Install the library:
Download [the release](https://github.com/xwiimote/xwiimote/releases/tag/xwiimote-2) and unzip:
$ tar -xJf [filename for release]

install:
$ sudo apt-get install automake
$ sudo apt-get install pkg-config
$ sudo apt-get install libudev-dev

run:
$ autoreconf
$ autoconf
$ ./configure~
note the tilde; this new configure is generated by autoreconf

$ make
$ sudo make install
this should put libxwiimote.so in usr/local/lib; if it doesn't, you can move it there manually

- Make a venv:
install pyenv
$ curl -fsSL https://pyenv.run | bash
create venv
$ pyenv virtualenv wiivenv
$ pyenv activate wiivenv


- After editing the CFFI, run pip install .

- run the file:
$ sudo -E python3 wiitest.py
